Шаг 1. Запуск бэкенда (и раздача фронтенда)
Фронтенд у нас отдаётся самим бэкендом (через папку 

miniapp
), поэтому тебе нужно запустить только один сервер — FastAPI. Открой терминал в корне проекта (/home/artemy/PycharmProjects/lingua_backend) и выполни:

bash
# 1. Активируем виртуальное окружение
source .venv/bin/activate
# 2. Запускаем сервер uvicorn (с флагом --reload для автоматической перезагрузки при изменениях в коде)
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
Если всё ок, в консоли появится Uvicorn running on http://0.0.0.0:8000.

Шаг 2. Запуск туннеля (чтобы VK увидел локальный сервер)
Так как ngrok ломает VK своим экраном-предупреждением, а Cloudflare блокируется твоим файрволом, мы используем бесплатный SSH-туннель localhost.run (или serveo.net). Не закрывая первый терминал, открой новую вкладку терминала и выполни:

bash
ssh -o StrictHostKeyChecking=accept-new -R 80:localhost:8000 nokey@localhost.run
В консоли появится текст, среди которого будет твоя новая HTTPS-ссылка. Она будет выглядеть примерно так: https://b5b36132298829.lhr.life

(💡 Важно: бесплатные туннели иногда обрываются, если долго нет запросов. В таком случае просто нажми Ctrl+C и запусти команду заново, чтобы получить новую ссылку).

Шаг 3. Настройка VK Mini App
Каждый раз, когда ты запускаешь новый туннель и получаешь новую ссылку, тебе нужно сказать ВКонтакте, куда отправлять пользователей.

Зайди в настройки своего приложения: Мои приложения VK -> Lingua
Перейди в раздел Настройки.
Найди поля URL для Web и URL мобильной версии.
Вставь туда свою ссылку из терминала и ОБЯЗАТЕЛЬНО добавь в конец /miniapp/. Пример: https://b5b36132298829.lhr.life/miniapp/
Нажми кнопку Сохранить в самом низу.
Шаг 4. Всё готово! 🚀
Открывай приложение ВКонтакте. Оно постучится по твоему новому адресу, пройдёт через туннель на твой локальный компьютер, uvicorn отдаст файлы фронтенда (HTML/CSS/JS), а потом начнёт успешно обрабатывать все запросы к API базы данных (/students/auth, /bookings/my и т.д.).

— Если будешь вносить изменения в код:

Python / Бэкенд: благодаря --reload сервер будет перезапускаться сам при сохранении файлов.
HTML / CSS / JS: просто обновляй страницу в приложении VK (иногда нужно свайпнуть вниз или переоткрыть приложение, чтобы сбросить кэш телефона).
